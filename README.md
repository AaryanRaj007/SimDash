## What it will do (when finished)

- Home screen widget with a circular data usage ring  
- Automatic carrier detection (Jio, Airtel, Vi, BSNL)  
- Smart alerts at 40%, 60%, and 80% usage  
- Plan validity countdown (with expiry reminders)  
- Daily morning summary notification  
- Works offline  all data stays on your device  
- Follows system dark/light mode  


## Current state

This project is in **early development**. Many parts are incomplete or unstable:

- Widget may show incorrect or stale data  
- USSD balance fetching is unreliable on some devices  
- Alert system is partially implemented  
- Onboarding flow is incomplete  
- UI screens are placeholders  
- Not tested across devices  

**Do not use this as your main data tracker yet.**

## Planned support

- Jio · Airtel · Vi · BSNL
